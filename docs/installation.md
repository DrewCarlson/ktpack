## Setup

### macOS

```shell
$ curl https://github.com/DrewCarlson/Ktpack/releases/latest/download/ktpack-macos.zip -o ktpack-macos.zip
$ unzip ktpack-macos.zip && cd ktpack-macos

$ ktpack version
Ktpack version 1.0.0-SNAPSHOT
```
